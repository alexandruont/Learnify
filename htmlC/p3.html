Lesson 3: Headings, Paragraphs, and Text Formatting

HTML provides various tags to structure and format text content on a webpage. Understanding these tags is essential for organizing information clearly.

Key Points:
  • <h1> to <h6> define headings, with <h1> being the largest.
  • <p> defines a paragraph of text.
  • <b> or <strong> make text bold, while <i> or <em> italicize it.
  • Comments are written using <!-- comment --> and are not displayed in the browser.
Example:
<h1>Main Title</h1>
  <p>This is a paragraph with <strong>bold</strong> and <em>italic</em> text.</p>
  <!-- This is a comment -->
