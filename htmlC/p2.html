Lesson 2: Basic Structure of an HTML Page

Every HTML page follows a basic structure made up of key elements that define the content and layout of the webpage.

Key Points:
  • <!DOCTYPE html> declares the document type.
  • <html> is the root element that wraps all content.
  • <head> contains meta information, the title, and links to styles/scripts.
  • <body> contains everything visible on the page (text, images, buttons).
Example:
<!DOCTYPE html>
  <html>
    <head>
      <meta charset="UTF-8">
      <title>Sample Page</title>
    </head>
    <body>
      <h1>Welcome!</h1>
      <p>This is an example of a basic HTML page.</p>
    </body>
  </html>
