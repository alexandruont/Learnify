Lesson 5: Lists and Tables

Lists and tables help you organize content clearly and make your HTML pages more structured and readable.

Key Points:
  • Use <ul> (unordered list) and <ol> (ordered list) to create lists, with <li> for each item.
  • Use <table> to create a table, with <tr> for rows, <th> for headers, and <td> for cells.
  • Tables are useful for displaying structured data such as schedules, prices, or comparisons.
Example:
<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
  </ul>

  <table border="1">
    <tr>
      <th>Name</th>
      <th>Age</th>
    </tr>
    <tr>
      <td>Alice</td>
      <td>24</td>
    </tr>
  </table>
