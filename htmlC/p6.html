Lesson 6: Forms and Inputs

HTML forms are used to collect user input. Forms are the foundation of user interaction in websites, allowing data submission, searches, login, and more.

Key Points:
  • The <form> tag defines a form element.
  • Common input elements include <input>, <textarea>, <select>, <button>.
  • Each input uses a type attribute to define its behavior (e.g., text, password, email).
  • The action attribute in <form> defines where to send the data, and method specifies how (GET/POST).
Example:
<form action="submit.html" method="post">
    <label>Name:</label>
    <input type="text" name="username" /><br/>
    <label>Password:</label>
    <input type="password" name="password" /><br/>
    <button type="submit">Login</button>
  </form>
