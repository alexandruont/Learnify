Lesson 1: What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It tells the browser how to display text, images, and other content on a webpage.

Key Points:
  • HTML uses tags to structure content (like <p> for paragraph, <h1> for largest header size, <img>).
  • HTML files are saved with a .html extension.
  • It forms the skeleton of all websites.
Example:
<!DOCTYPE html>
  <html>
    <head>
      <title>My First Webpage</title>
    </head>
    <body>
      <h1>Hello, world!</h1>
      <p>This is my first webpage.</p>
    </body>
  </html>
